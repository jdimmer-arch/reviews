## Performance Review Process

### Step 1: Detailed Review Consolidation

Each team member has a comprehensive consolidation document that includes:

- Historical performance data (2022-2025)
- STAR format assessments for all competencies
- Manager comments and ratings for each area
- Development trajectory and promotion readiness

### Step 2: Manager's Rating Form Generation

After completing detailed consolidations, generate Manager's Rating Forms using:

#### Rating Categories (1-5 scale, decimals allowed)

1. **Consulting Skills**: Technical competency, problem-solving, business acumen, innovation
2. **Professional Skills**: Communication, project management, self-management, people development  
3. **Client Delivery**: Delivery quality, client engagement, accountability, time management
4. **Contribution to Archetype**: IP development, financial responsibility, sales/marketing support
5. **Overall Performance**: Holistic assessment of total contribution and growth

#### Key Performance Questions

- **What went well?** (3 accomplishments in 2025)
- **What didn't go well?** (3 challenges in 2025)  
- **What will be the focus?** (3 focus areas for rest of 2025)

#### Manager's Notes

- Performance summary with key strengths and development areas
- Overall assessment and recommendations

### Step 3: Post-Review Requirements

See **Post_Review_Requirements.md** for tasks to be completed after review finalization.

## Usage

1. Complete individual review consolidations using detailed competency assessments
2. Use **Competency_Mapping_Guide.md** to map detailed ratings to summary categories
3. Generate Manager's Rating Forms using **Manager_Rating_Form_Template.md**
4. Extract key accomplishments, challenges, and focus areas from existing reviews
5. Prepare final rating forms for copy/paste into official review systems

## File Naming Convention

- Consolidation documents: `Consolidation [Employee Name].md`
- Manager rating forms: `Manager_Rating_Form_[Employee Name].md`
- Historical reviews: `YYYY_Review_[EmployeeName].pdf` or `YYYY_MY_Review_[EmployeeName].pdf`
