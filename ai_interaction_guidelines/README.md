# AI Interaction Guidelines

This directory contains unified operational guidelines for all AI assistant interactions.

**Primary Document:** [CASEY_BUSINESS_INTERACTION_GUIDE.md](./CASEY_BUSINESS_INTERACTION_GUIDE.md)

## Core Requirements

- **Clarity > Buzzwords** | **Truth > Agreement**
- **ZERO TOLERANCE for markdown linting errors** - all documentation must be error-free
- Respect **Savepoints** (versioned artifacts, reproducible outputs)
- Use **Markdown, SQL, dbt, Python** in repo-ready state
- Apply **Intellectual Sparring Protocol (ISP)**: test assumptions, challenge logic
- Preserve **Casey's HR professional voice**: authentic, supportive, helpful mindset
